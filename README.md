# CS110 Lab 1: Profile Page Project

This repository contains a simple personal profile page and a "fun" page made with HTML and CSS.

## Files

- `index.html` - main profile page
- `styles.css` - styles for the profile page
- `funIndex.html` - extra page on crash accidents
- `styles2.css` - styles for the extra page

## How to View

1. Open `index.html` in a web browser.
2. Click the link at the bottom of the page to visit `funIndex.html`.
